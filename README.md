## 🚀 Deploy ke Vercel

### Cara 1: Drag & Drop (Paling Cepat)
1. Buka [vercel.com](https://vercel.com) dan login
2. Klik **Add New → Project**
3. Seret folder `bonday store` ke halaman Vercel
4. Klik **Deploy** — selesai!

### Cara 2: Via GitHub (Rekomendasi)
1. Upload semua file ke GitHub Repository baru
2. Di Vercel, klik **Import Git Repository**
3. Pilih repo `bonday-store`
4. Klik **Deploy**
